    • wmsclient
      • In der Datei index.php sind für eine Installation die Pfade zum WMS Proxy und AAAD anzupassen
    • wmsproxy
      • In der Datei config.php sind für eine Installation die Pfade zum AAAD sowie ggf. die OAuth Secrets anzupassen

      • Verzeichnis /cache/templates_c muss beschreibbar sein
      • Verzeichnis /log muss beschreibbar sein
      • MySQL Struktur mit Beispieldaten liegt in Datei aaad_mysql_dump.sql vor
      • MySQL Einstellungen in Datei core/init.php in Zeile 67 anpassen
